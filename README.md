# Kritika Singh - Portfolio

Modern portfolio website showcasing full-stack development skills and projects.

## Features
- Responsive design with Tailwind CSS
- Interactive contact form
- Project showcase
- Skills visualization
- Professional experience timeline

## Tech Stack
- React 19
- Tailwind CSS
- Vite
- EmailJS integration

## Setup
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
